Built and evolved over 6+ years to manage real-money trading operations, compliance, and broker growth.

## 🛠 Tech Stack
- Backend: Laravel (PHP)
- Frontend: Blade, Bootstrap, SCSS
- Database: MySQL
- Trading: MT5 API integration
- Compliance: KYC (manual & automated)
- Payments: Multiple gateways
- Cron Jobs,Queues
